# Laura AI - Real Estate Assistant

This contains everything you need to run the Laura AI Real Estate Assistant app locally.

Laura AI is an intelligent real estate assistant that helps agents with content generation, lead management, and workflow automation.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
